A project to share photos easily by category.

Frontend:

Vue.js, Vuex
Bootstrap
Webpack (custom config)
Jest + Avoriaz for testing

Backend
Node.js (hosted on Amazon EC2)
MongoDB (hosted on Mlab)
Image Storage on Amazon S3
